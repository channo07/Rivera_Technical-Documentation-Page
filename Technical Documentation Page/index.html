HTML Basics
Introduction Elements Attributes Forms Media
Introduction

HTML (HyperText Markup Language) is the standard language for creating webpages.

It allows developers to structure content on the web using elements such as headings, paragraphs, lists, and more.

HTML is the backbone of web development, working together with CSS and JavaScript.

Elements

HTML documents are made up of elements, defined by tags.

For example:

<p>This is a paragraph.</p>

Common elements include:

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a>
Attributes

HTML elements can have attributes, providing additional information about the element.

Attributes are always specified in the opening tag and usually come in name-value pairs:

<a href="https://example.com">Visit Example</a>

Common attributes include:

  • href (for links)
  • src (for images)
  • alt (for images)
Forms

HTML forms allow users to interact with a website by submitting data.

For example:

<form>
  <input type="text" name="username">
  <input type="submit">
</form>

Forms include input elements, buttons, and other controls for user input.

Media

HTML supports embedding multimedia such as images, videos, and audio files.

Examples:

<img src="image.jpg" alt="An example image"> <video controls><source src="video.mp4" type="video/mp4"></video>

Proper use of media enhances the user experience.
